Frequently Asked Questions

What image formats are supported?

We support JPG, PNG, and WEBP formats. The output is always a PNG file with transparent background.

Is there a file size limit?

Yes, the maximum file size is 10MB for free users and 25MB for Pro/Business users.

How long does processing take?

Processing typically takes 3-5 seconds depending on image complexity and server load.

Can I remove background from multiple images at once?

Yes, batch processing is available for Pro and Business subscribers (up to 10 images at once).
